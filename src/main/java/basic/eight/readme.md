2.二叉树的结构定义如下:
    class Node{
        v value;
        Node left;
        Node right;
        Node parent;
    }
    给你二叉树的某一个节点,返回该节点的后继节点。
    后继节点：是指二叉树中序遍历的，node的下一个节点叫做后继节点。 
    思路：因为是中序遍历后面的节点，所以如果一个节点有右孩子的话， 
    如果一个节点没有右孩子的话，先找到他的父节点，如果该节点是其其父节点的左孩子那么父节点就是后继节点，如果不是的话就继续往上找。
